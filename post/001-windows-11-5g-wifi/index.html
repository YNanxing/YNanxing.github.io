Windows 11 如何使用 5G 频段热点分享网络

· 2022-10-24 · # Windows

引言

电脑安装 Windows 11 后，发现无法开启 5 GHz 频段的移动热点，在网上摸索一段时间后，终于在微软社区发现了解决方法。

此问题原因是由于 Intel 响应国家的监管限制而进行的设定，虽然 Intel 提供了相应的解决方法，但是由此产生的问题将由您承担。

网页链接：点击进入网页

第一步：打开设备管理器

在 Windows 11 自带的搜索栏搜索「设备管理器」，并打开

第二步：更改网卡设置

在设备管理器中打开网络适配器，双击你的 intel 网卡，在高级选项中找到 802.11a/b/g 无线模式，选择值为 5GHz 频段。

第三步：打开 5G 热点

在右下角 Wifi 处单击打开快速设置，右键点击「移动热点」，单击「转入设置」打开移动热点设置页面。

或者直接在搜索栏搜索「移动热点」。

在移动热点设置中选择 5GHz。

打开热点，完成！

注意

  • 不知道什么原因，打开热点时需要把 Clash 代理节点设置为直连。
  • 如果没有效果，建议更改为原来的设置，然后再去搜索其他方法。
