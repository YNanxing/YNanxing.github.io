Navicat 教育版申请

· 2022-12-29 · # 数据库 # 软件 # 教育版

什么是 Navicat？

Navicat 是香港卓软数字科技有限公司生产的一系列MySQL、MariaDB、MongoDB、Oracle、SQLite、PostgreSQL 及 Microsoft SQL Server 的图形化数据库管理及发展软件。它有一个类似浏览器的图形用户界面，支持多重连线到本地和远程数据库。

第一步：填写 NAPP 申请表

在浏览器中打开 Navicat 教育版申请官网：
申请网站链接：https://www.navicat.com.cn/sponsorship/education/student

打开网站后会有一个「Navicat 学术伙伴计划（NAPP）- 学生 - 申请表」，按照表中的内容填写即可。

第二步：接收邮件

大概 1-2 天，在填写的教育邮箱内会接收到 Navicat 的邮件和一个申请表格。

第三步：学信网申请学籍验证信息

打开学信档案：学信档案官网

点击「学籍」

点击「查看该学籍的在线验证报告」

点击「申请」申请在线验证报告

第三步：填写表格

截图「教育部学籍在线验证报告」，填写到「Navicat 教育版申请信息」表格中。

第四步：回复邮件，把填写的表格添加到附件中

回复 Navicat 邮件后，会再次接收到一封邮件

等待申请完成即可！
